# Content
The folder contains 
1. STA141B_Final_Project.ipynb: The python Juypter Notebook fle contains the whole project workflow
2. Two python Pandas data frame extracted from the web-scraping process, cleaned already
    - df_ev.csv: The main dataset that we used for the STA141B_Final_Project.ipynb 
    - df_ev_geo.csv: Dataset that not yet used, but is useful for further in-depth analysis
3. Three Interactive Maps
    - ev_charging_station_map.html
    - tesla_level_charger_map.html
    - tesla_fast_charger_map.html
